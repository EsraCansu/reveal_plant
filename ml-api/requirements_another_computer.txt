# ===================================================================
#
# WINDOWS:
# --------
# 1. Anaconda Prompt aç
# 2. Proje klasörüne git:
#    cd C:\path\to\reveal_plant\ml-api
#
# 3. Conda environment oluştur:
#    conda create -n myenv python=3.10 -y
#
# 4. Aktifleştir:
#    conda activate myenv
#
# 5. Tüm paketleri yükle:
#    pip install -r requirements_another_computer.txt
#
# 6. FastAPI sunucusunu başlat:
#    python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
#
# 7. Kontrol et:
#    curl http://localhost:8000/health
#
# ========================================================================
#
# LINUX / MACOS:
# ---------------
# 1. Terminal aç
# 2. Proje klasörüne git:
#    cd /path/to/reveal_plant/ml-api
#
# 3. Conda environment oluştur:
#    conda create -n myenv python=3.10 -y
#
# 4. Aktifleştir:
#    conda activate myenv
#
# 5. Tüm paketleri yükle:
#    pip install -r requirements_another_computer.txt
#
# 6. FastAPI sunucusunu başlat:
#    python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
#
# 7. Kontrol et:
#    curl http://localhost:8000/health
#
# ========================================================================
#
# MODEL DOSYASINI EKLE:
# ---------------------
# ml-api/model/ klasörüne şu dosya olması gerekir:

#

# DOCKER'DA ÇALIŞTIRACAKSAN:
# --------------------------
# Docker desktop yüklü olmalı, sonra:
#
# 1. Docker compose up:
#    docker-compose up -d ml-api
#
# 2. API test et:
#    curl http://localhost:8000/health
#
# ========================================================================
#
# TROUBLESHOOTING:
# ----------------
#
# 1. "ModuleNotFoundError: No module named 'tensorflow'"
#    → pip install --upgrade tensorflow keras
#
# 2. "CUDA not found" (GPU kullanacaksan)
#    → tensorflow-gpu yükle: pip install tensorflow-gpu
#
# 3. "Model bulunamadı"
#    → model/PlantVillage_Resnet101_FineTuning.keras dosyasını kontrol et
#
# 4. "Port 8000 zaten kullanılıyor"
#    → Farklı port kullan: --port 8001
